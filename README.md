# Build Locally
1. Install  and setup Ruby according to [Ruby instructions](https://jekyllrb.com/docs/installation/#requirements)
2. Install Jekyll according to these [instructions](https://jekyllrb.com/docs/)
```
$  git clone https://github.com/TheNAPPLab/TheNAPPLab.github.io.git
$ cd TheNAPPLab.github.io
$ bundle install
$ bundle exec jekyll serve --watch --incremental
```

This should start the web server locally, and enables incremental build.
